# ML Engineering

Quick Apple Silicon environment check for PyTorch MPS and Apple MLX.

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

## Run

```bash
python test_env.py
```

You can also run it directly:

```bash
./test_env.py
```
